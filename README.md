# jaxgardner.github.io

Personal portfolio site for Jaxon Gardner — software engineer building Angular web apps and Flutter mobile experiences.

Live at: <https://jaxgardner.github.io>

## Editing

This is a plain static site — no build step.

- `index.html` — page structure and content
- `styles.css` — all styling

Push to `main` and GitHub Pages will deploy automatically.

## Adding a project

Copy an existing `<article class="card">` block in `index.html`, drop it into the appropriate section (`#angular` or `#flutter`), and edit the name, description, tech stack pills, and links.
